05.04.

Сви студенти који полажу још неки колоквијум у недељу 14. априла, поред колоквијума из Објекто-оријентисаног програмирања, потребно је да пошаљу мејл на anja_bukurov@matf.bg.ac.rs како би били распоређени у групу која се не поклапа са другим колоквијумом. **Рок за јављање је недеља 07. април.** Распоред студената по групама и учионицама биће објављен након истека рока.

03.04.

### Колоквијум 2019.

Градиво за колоквијум из Објекто-оријентисаног програмирања обухвата лекције закључно са интерфејсима и енумерационим типовима, не укључујући клонирање.

Колоквијум ће бити одржан у недељу 14. априла. Распоред студената по групама и учионицама биће накнадно објављен.
